# Agent instructions

## Git commits

- NEVER add advertising or attribution trailers to commit messages. In
  particular, do NOT append `Co-authored-by: Cursor <cursoragent@cursor.com>`
  (or any similar "Co-authored-by", "Generated by", or tool-attribution line)
  to any commit. Commit messages must contain only the human-authored subject
  and body describing the change.
- If the environment adds such a trailer automatically, strip it before the
  commit is finalized (e.g. amend the message) so it never reaches history.
- Write concise, descriptive messages focused on the "why" of the change.
